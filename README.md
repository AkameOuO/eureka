# Eureka 收集追蹤器

一個簡潔而優雅的收集追蹤應用，幫助你管理和追蹤遊戲裝備（Eureka）的收集進度。

## ✨ 功能特性

### 🎯 核心功能
- **進度追蹤** - 實時查看你的收集進度和完成百分比
- **智能篩選** - 按稀有度、區域快速篩選裝備
- **隱藏已完成** - 只顯示還未完成的裝備，不再混亂
- **雙語支持** - 支持繁體中文和英文界面
- **本地儲存** - 自動保存你的收集紀錄，無需登入

### 📊 資料備份
- **匯出收集紀錄** - 隨時備份你的進度
- **匯入舊資料** - 恢復之前的收集紀錄
- **自動驗證** - 系統自動檢查數據有效性

### 🎨 使用者體驗
- **優雅設計** - 粉紅主題配合清爽界面
- **響應式設計** - 同時適配桌機和行動設備
- **直覺操作** - 無需教學，一目瞭然

## 🚀 快速開始

### 在線使用（推薦）
直接訪問部署的應用，無需安裝任何東西。

### 本地運行

#### 前置條件
- 已安裝 Node.js（v16 或更新版本）

#### 安裝步驟
```bash
# 1. 進入項目目錄
cd eureka

# 2. 安裝依賴
npm install

# 3. 啟動開發服務器
npm run dev
```

訪問 `http://localhost:5173` 即可使用。

## 📖 使用指南

### 基本操作

#### 1. 追蹤進度
- 在表格中勾選複選框標記已收集的裝備顏色和部位
- 頂部的進度條會實時更新你的完成百分比
- 左側邊欄顯示詳細的進度統計

#### 2. 篩選裝備
**按稀有度篩選：**
- 點擊左側的 `3⭐`、`4⭐`、`5⭐` 按鈕
- 只顯示選中稀有度的裝備

**按區域篩選：**
- 點擊表格上方的區域標籤
- 在「心願原野」和「伊贊之土」之間切換

**隱藏已完成：**
- 勾選「隱藏已完成」選項
- 只顯示還有未收集部位的裝備

#### 3. 切換語言
- 點擊右上角的語言按鈕
- 選擇「English」或「繁體中文」

### 數據管理

#### 🔄 匯出（備份進度）
1. 點擊左側的「**匯出**」按鈕
2. 在彈出視窗中選擇：
   - **複製** - 複製 JSON 內容到剪貼板
   - **下載** - 下載為 `.json` 文件保存
3. 關閉視窗

#### 📥 匯入（恢復進度）
1. 點擊左側的「**匯入**」按鈕
2. 選擇匯入方式：
   - **上傳文件** - 選擇之前下載的 JSON 文件
   - **貼上內容** - 直接貼上 JSON 文本（來自複製）
3. 系統會驗證數據有效性
4. 導入成功後頁面會自動刷新

## 💾 數據說明

### 自動保存
- 你的所有操作都會自動保存到瀏覽器的本地儲存
- 即使關閉瀏覽器也不會遺失
- 清空瀏覽器數據會清除保存的紀錄

### 雲端備份
- 本應用不提供雲端同步功能
- 請定期使用「匯出」功能備份你的進度
- 重要的進度打開新裝置時記得「匯入」

### 支持的語言
目前支持：
- 🇹🇼 繁體中文
- 🇬🇧 English

## ❓ 常見問題

### Q: 我的進度消失了怎麼辦？
A: 請嘗試以下方案：
1. 重新加載頁面
2. 確認瀏覽器是否啟用了 localStorage
3. 清空瀏覽器緩存後重新打開

### Q: 可以在多個設備間同步進度嗎？
A: 可以的！請使用匯出/匯入功能：
1. 在設備 A 上匯出進度
2. 在設備 B 上匯入備份文件
3. 完成！

### Q: 為什麼某些裝備沒有顯示？
A: 可能的原因：
- 已被篩選器過濾（檢查稀有度或區域篩選）
- 已被「隱藏已完成」選項隱藏
- 嘗試清除所有篩選器重試

### Q: 支持多語言嗎？
A: 目前支持繁體中文和英文。點擊右上角的語言按鈕即可在兩種語言間切換。

### Q: 我的建議要寄給誰？
A: 非常歡迎你的建議！請聯絡項目維護者分享想法。

## 🛠 故障排除

### 應用無法加載
- 確認瀏覽器是否支持 JavaScript
- 嘗試在另一個瀏覽器測試
- 清空瀏覽器緩存

### 導入失敗
- 確認導入的文件格式正確（應為 JSON）
- 嘗試重新複製內容並檢查格式
- 確認 JSON 包含 `"collection"` 字段

### 進度沒有保存
- 檢查瀏覽器是否允許 localStorage
- 確認帳戶有足夠的存儲空間
- 嘗試清空瀏覽器緩存後重新操作
