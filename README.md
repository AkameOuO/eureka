# Eureka 收集追蹤器

一個為《無限暖暖》祝福閃光設計的收集進度工具。

你可以用它快速記錄每個祝福閃光的顏色與部位收集狀態，並透過 Google Drive 進行雲端同步與多裝置延續。

## 功能總覽

### 核心追蹤
- 以表格方式管理所有祝福閃光收集狀態。
- 進度條即時顯示已收集數與總數。
- 支援區域切換與完整資料瀏覽。

### 篩選與搜尋
- 稀有度篩選（3/4/5 星）。
- 名稱搜尋（中英文）。
- 標籤篩選。
- 隱藏已完成項目。

### 本地資料管理
- 所有操作自動保存到瀏覽器本地儲存。
- 支援匯出 JSON（複製或下載）。
- 支援匯入 JSON（貼上或上傳檔案）。
- 會顯示本地存檔最後更新時間。

### Google Drive 雲端同步
- Google 登入授權後，可使用雲端存檔管理。
- 支援建立空存檔、建立並上傳、覆蓋、套用、重新命名、刪除。
- 可顯示可用存檔清單與目前套用中的存檔。
- 登入逾時時，介面會提示重新登入。
- 已登入狀態可切換帳號。

### 多視窗與狀態同步
- Collection 與登入狀態支援跨分頁同步。
- 透過 `storage`、`visibilitychange`、`pageshow` 降低多視窗與手機閒置恢復時的狀態漂移。

### 介面與可用性
- 支援繁體中文與英文切換。
- 響應式版面，桌機與手機皆可使用。
- 手機版提供底部抽屜式篩選器。

## 使用方式

### 1. 追蹤收集進度
1. 在表格勾選已收集的顏色/部位。
2. 上方進度條會即時更新。

### 2. 篩選資料
1. 使用左側（或手機抽屜）篩選器。
2. 可同時套用稀有度、名稱、標籤與隱藏已完成。

### 3. 備份與還原
1. 進入「雲端同步/匯入匯出」視窗。
2. 在匯出頁籤備份 JSON。
3. 在匯入頁籤貼上或上傳 JSON 還原。

### 4. 使用雲端同步
1. 在同步頁籤登入 Google。
2. 依需求建立或選擇雲端存檔。
3. 使用套用/覆蓋/重新命名/刪除等操作管理存檔。

## 資料與隱私說明

- 本地資料保存在瀏覽器 localStorage。
- 啟用雲端同步後，資料存放於你的 Google Drive `appDataFolder`（應用程式專用資料夾）。
- 若清除瀏覽器資料，本地紀錄會遺失；建議定期匯出或使用雲端同步。

## 常見問題

### 進度看起來不同步？
- 請先確認是否在另一個分頁操作過。
- 切回頁面後會自動嘗試同步狀態。
- 若仍異常，可重新整理頁面再確認。

### 登入後為什麼又顯示未登入？
- 可能是 token 已逾時，請重新登入。
- 也可能是你在其他分頁登出，狀態會同步到目前頁面。
