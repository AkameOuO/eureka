隱私權政策

最後更新時間：2026 年 4 月

概述

本應用程式（「本應用程式」）是一款個人收集追蹤工具。我們致力於保護你的隱私。本政策說明收集的資料、使用方式及你的權利。

我們收集的資料

應用程式資料

本應用程式將你的遊戲收集記錄（個人資料及收集進度）直接儲存在你自己的 Google Drive 中，使用受限的 drive.appdata 範圍。此資料：

  • 專門儲存在你的個人 Google Drive 中
  • 永遠不會傳輸到或儲存在我們的伺服器上
  • 只能由你和本應用程式存取

我們不收集、處理或儲存任何個人資訊在我們自己的伺服器上。

Google 授權資料

本應用程式只要求儲存和同步你的收集資料所需的 Google Drive drive.appdata 權限。我們不要求 Google 帳戶個人資料存取權限，如你的電子郵件、名稱或個人相片。

為了在使用期間保持登入狀態，本應用程式將 OAuth 存取令牌和令牌逾期時間儲存在你的瀏覽器本地端。此驗證資料永遠不會傳輸到或儲存在我們的伺服器上。

分析資料

本網站使用 Cloudflare Web Analytics 來收集匿名流量統計，包括：

  • 頁面瀏覽量和造訪持續時間
  • 來源 URL 和地理區域（國家等級）
  • 裝置類型和瀏覽器類型

Cloudflare Web Analytics 不使用 Cookie，且 不識別個人使用者。所有資料都是彙總且匿名的。如需詳細資訊，請參閱 Cloudflare 隱私權政策。

Google API 使用

本應用程式使用 Google Drive API 搭配 drive.appdata 範圍，專門用於儲存和同步你的遊戲收集資料。我們不會存取你 Google Drive 中的任何其他檔案。

我們對 Google API 的使用符合 Google API 服務使用者資料政策 ，包括受限使用要求。

資料分享

我們不會向任何第三方出售、分享或披露你的資料。你的收集資料保留在你自己的 Google Drive 中，永遠不會被我們存取。

資料刪除

要刪除你的應用程式資料，你可以：

  1. 在 Google 帳戶權限頁面 上移除應用程式的存取權限
  2. 直接從你的 Google Drive appdata 資料夾中刪除應用程式檔案

本政策的更改

我們可能不時更新本政策。任何變更將反映在本頁面上，並在頂部更新日期。
